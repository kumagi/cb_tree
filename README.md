# cb_tree
A crit-bit tree implementation for C++

## Features
- Header-only  
  `#include <cb_tree/cb_tree.hpp>`
- STL-like Containers
  - cb_set (like std::set)
  - cb_map (like std::map)

## Requirements

### Supported Compilers
- Clang (>= 3.7.0)
- Visual Studio 2015

### External Libraries
- [Catch](https://github.com/philsquared/Catch) (for test)
