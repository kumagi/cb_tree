# cb_tree
Crit-bit tree for C++

## Features
- Header-only  
  `#include <cb_tree/cb_tree.hpp>`

## Requirements

### Supported Compilers
- Clang (>= 3.7.0)
- Visual Studio 2015

### External Libraries
- [Catch](https://github.com/philsquared/Catch) (for test)
